# UI Copy Guide

MaoqiuPlayer 的用户界面应始终像一款普通本地媒体播放器。

## 推荐用语

- 毛球播放器
- MaoqiuPlayer
- 首页
- 最近播放
- 本地视频
- 本地图片
- 播放列表
- 打开文件
- 打开媒体
- 媒体库
- 设置
- 高级设置
- 高级工具
- 媒体包
- 私有媒体包
- 生成媒体包
- 导入媒体包
- 查看媒体包信息
- 校验媒体包
- 缓存管理
- 数据库维护
- 文件完整性校验
- 访问口令

## 不建议出现在首页或主导航的用语

- 底层存储实现
- 访问控制实现
- 文件保护实现
- 媒体包底层格式
- 开发者调试信息

这些内容如需说明，应放在 README 靠后的高级功能或安全说明章节，或放在开发文档中。

## 页面口吻

首页和媒体库页面应短句、播放器化、任务导向：

- “继续上次观看”
- “浏览已导入视频”
- “查看本地图片”
- “直接打开本地媒体”

高级工具页面应保持克制：

- “生成、导入、查看和校验媒体包”
- “移除临时播放缓存”
- “刷新媒体库索引和记录”

## 技术文档口径

技术文档可以说明 `.mqp` 媒体包数据采用加密存储，并说明 Magic、版本、KDF、cipher 等字段。安全声明不放在 README 开头，应位于高级功能之后。
