- 补全 Android APK 功能：首页、最近播放、本地视频、本地图片、播放列表、媒体库和设置入口已可在移动端使用。
- Android 支持应用内视频播放，包括播放/暂停、系统控制条、上一项/下一项、倍速和全屏入口。
- Android 支持应用内图片查看，包括双指缩放、拖动查看、适应窗口、实际大小、上一张/下一张和旋转入口。
- Android 支持扫描本机视频和图片，并可导入文件到本地媒体库。
- Android 支持打开 `.mqp` 私有媒体包，后台完成识别、校验、临时解包并播放包内媒体。
- macOS 发布包已改为安装包形式：打开 DMG 后双击 `MaoqiuPlayer-macOS.pkg` 安装到 `/Applications`，不再是直接拖出 app 打开。
- 同步更新 Windows、Linux、Android、iOS 的版本号和在线打包产物到 `0.1.5`。
